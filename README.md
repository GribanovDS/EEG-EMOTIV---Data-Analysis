# EEG EMOTIV - Data Analysis

Этот ноутбук реализует полный цикл обработки данных ЭЭГ и эмоциональных метрик с нейроинтерфейса EMOTIV EPOC X в задаче восприятия инфоповодов

Первая часть ноутбука отвечает за подготовку и предобработку ЭЭГ: чтение CSV с учётом разных разделителей, синхронизацию сырых сигналов с маркерами по глобальному времени (Timestamp - timestamp), вырезку эпох вокруг стимулов (предстимульный и постстимульный интервалы), базовую коррекцию и усреднение по эпохам и респондентам. На основе усреднённых данных для 14 каналов создаётся объект MNE EvokedArray, строятся ERP‑графики и топографические карты, которые автоматически сохраняются в каталог `ERP_RESULTS`.

Вторая часть посвящена эмоциональным метрикам EMOTIV с низкой частотой обновления (~0.1 Гц). Ддля каждого стимула формируются длинные окна до/во время/после предъявления, выполняется интерполяция и усреднение по респондентам. Итогом являются графики динамики эмоциональных показателей по каждому стимулу, сохраняемые в отдельную папку (например, `EMO_RESULTS`, `EMO_RESULTS_MALE, `EMO_RESULTS_FEMALE`).

В ноутбуке также предусмотрены базовые проверки и обработка типичных проблем: неоднородные разделители в CSV (, и ;), пропуски (NaN) в эмоциональных метриках.

> Тестировано с Python 3.10, numpy ≥ 1.24, pandas ≥ 2.0, mne ≥ 1.5, matplotlib ≥ 3.7
